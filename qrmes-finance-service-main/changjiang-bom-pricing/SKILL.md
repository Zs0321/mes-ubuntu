# Changjiang BOM Pricing

## 适用范围

本 skill 面向电机或电控相关配件的 BOM 报价，优先理解以下场景：

- 定子、转子、铁芯、绕组、磁钢、轴、端盖、机壳
- 接线座组件、接线盒组件、端子座组件、连接器、线束
- 轴承、油泵、过滤器、导电环、法兰、堵头、弹簧、密封件、透气阀
- 电机总成、电控配件、半成品组件、多材质复合组件

当名称或规格存在歧义时，优先按电机/电控供应链中的常见配件理解，不要泛化到无关行业的通用零件。

## 当前运行主流程（2026-04-18）

### 1. 复合组件合并规则

当 `物料编码` 相同且 `物料名称` 相同，但拆成多条不同 `材质` 行时，默认按“同一组件由多种材料组成”处理。

处理要求：

- 材料价格按各子项分别计价，再汇总为组件材料成本。
- 组件展示时，将材质按子项顺序去重拼接，例如：`铁芯27SW1400 + 漆包扁铜线`。
- 组件重量按各子项重量求和。
- 组件规格按子项规格去重后拼接。
- 数量保持组件数量，不把子项数量简单累加成翻倍数量。
- 工艺视为该复合组件的共同工艺；如果只有部分子项填写了工艺，则沿用已填写工艺作为组件工艺。
- 日志和说明中必须披露：`已按相同物料编码+物料名称合并多材质子项，按复合组件报价`。

典型场景：

- 定子组件 = 铁芯 + 漆包线
- 转子组件 = 磁钢 + 轴芯 + 端环
- 线束组件 = 线材 + 护套 + 端子
- 接线座组件 = 注塑壳体 + 金属端子 + 紧固件

### 2. 两阶段 AI 推断 + skills 报价主链路

只要 `工艺 / 材质 / 重量` 任一缺失，就要优先执行两阶段 AI 推断流程：

1. 第一阶段：AI 预推断
   - 输出 `process_guess`
   - 输出 `material_guess`
   - 对重量强相关器件且缺重量时输出 `estimated_weight_kg`
   - 输出 `confidence`
2. 第二阶段：将高置信度推断结果回填成 skills 的临时输入
   - `工艺 / 材质` 的高置信度门槛为 `0.60`
   - `重量` 的高置信度门槛为 `0.60`
3. 使用回填后的临时输入，先跑 skills 脚本报价
4. 最后再做第二轮正式 AI 报价补充说明或兜底

执行要求：

- 工艺缺失时，高置信度 `AI推断工艺` 可以作为 skills 的临时工艺输入。
- 材质缺失时，高置信度 `AI推断材质` 可以作为 skills 的临时材质输入。
- 对重量强相关器件，重量缺失时，高置信度 `AI推断重量` 可以作为 skills 的临时重量输入。
- 回填字段必须标记为 `AI推断参考（AI计算，用于本次AI报价）`，不能伪装成原始 BOM 主数据。
- 如果第二轮正式 AI 返回的 `process_guess / material_guess` 与第一轮预推断不一致，则以第二轮为准。
- 最终说明中必须写明：
  - 是否触发了两阶段 AI
  - 哪些字段来自 AI 推断
  - 推断置信度
  - 是否发生了第二轮覆盖第一轮

### 2.1 电机训练文档回填优先级

如果系统内存在电机专项训练文档（例如柳工双12/15/20电机 BOM 训练表）并已结构化为参考数据，则在进入两阶段 AI 前，先尝试按如下优先级回填：

1. `物料编码` 精确命中
2. `物料名称 + 规格` 精确命中
3. 仅在名称唯一且无歧义时，允许按 `物料名称` 单独命中

可回填字段：

- `材质`
- `重量`
- `工艺`
- `当前参考单价`（仅作为训练参考或 Excel 原始参考，不得当作 AI 锚点）

执行要求：

- 回填命中后，必须在结果中保留“训练参考命中”标记，说明来源的产品与 sheet。
- 训练文档回填优先于“相近物料估重”，因为它更接近电机真实项目主数据。
- 如果训练文档回填和 AI 预推断冲突，且训练文档字段完整，则优先以训练文档为准，再让第二轮 AI 补说明。

### 3. 电机/电控名称型物料价格区间

名称规格推断报价默认按电机/电控配件语境处理，以下区间均为 `单件价`：

- 轴承：`20 - 80 元/件`
- 油泵：`250 - 950 元/件`
- 换热器：`180 - 700 元/件`
- 过滤器 / 滤芯：`15 - 80 元/件`
- 导电环：`45 - 120 元/件`
- 连接器 / 插头 / 插座：`20 - 80 元/件`
- 接线座组件 / 接线盒组件 / 端子座组件：`120 - 900 元/件`
- 密封件 / 密封圈 / 油封 / 密封垫 / O型圈：`2 - 20 元/件`
- 透气阀 / 防水透气阀：`10 - 60 元/件`
- 线束：`18 - 120 元/件`
- 波形弹簧 / 弹簧：`8 - 35 元/件`

规则要求：

- 区间是单件价约束，整行金额仍按 `单价 × 数量` 计算。
- 如果 AI 返回价格落在区间外，按上下限修正。
- 如果 AI 未返回有效价格，但名称规格已能识别类别，可先按区间中值形成临时 `名称规格推断报价`，并标记待复核。
- 不允许使用 `Excel采购参考 / 金蝶采购参考 / 目标价参考` 作为 AI 报价保底锚点。

### 4. 电机/电控名称型物料重量区间

以下区间为缺重量时的校正和默认估重参考：

- 轴承：`0.08 - 1.80 kg`，默认 `0.35 kg`
- 油泵：`0.80 - 6.50 kg`，默认 `2.20 kg`
- 换热器：`0.50 - 8.00 kg`，默认 `2.80 kg`
- 过滤器 / 滤芯：`0.03 - 1.50 kg`，默认 `0.18 kg`
- 导电环：`0.08 - 1.20 kg`，默认 `0.25 kg`
- 连接器 / 插头 / 插座：`0.02 - 0.80 kg`，默认 `0.12 kg`
- 接线座组件 / 接线盒组件 / 端子座组件：`0.30 - 3.00 kg`，默认 `1.20 kg`
- 密封件 / 密封圈 / 油封 / 密封垫 / O型圈：`0.002 - 0.12 kg`，默认 `0.02 kg`
- 透气阀 / 防水透气阀：`0.005 - 0.08 kg`，默认 `0.02 kg`
- 线束：`0.05 - 3.50 kg`，默认 `0.45 kg`
- 波形弹簧 / 弹簧：`0.01 - 0.25 kg`，默认 `0.06 kg`
- 法兰 / 轴法兰：`0.08 - 3.00 kg`，默认 `0.45 kg`
- 转子挡板：`0.08 - 0.80 kg`，默认 `0.22 kg`
- 轴套 / 转子轴套：`0.05 - 0.60 kg`，默认 `0.18 kg`
- 压盖 / 轴承压盖：`0.05 - 0.80 kg`，默认 `0.20 kg`
- 上盖板 / 后盖板 / 三相盖板：`0.05 - 1.20 kg`，默认 `0.28 kg`
- 接线板 / 端子板：`0.05 - 0.60 kg`，默认 `0.18 kg`
- 连接器 / 插件保护罩 / 端子 / 格兰头：`0.01 - 0.30 kg`，默认 `0.06 kg`
- 堵头 / 橡胶堵头：`0.01 - 0.30 kg`，默认 `0.05 kg`
- 内六角 / 螺母 / 螺钉 / 螺栓 / 螺杆：`0.003 - 0.15 kg`，默认 `0.02 kg`

规则要求：

- 如果 `材质 = 未识别`，不能继续走“同材质相近物料估重”。
- 如果相近物料估重明显落在合理区间外，按类别默认重量修正。
- 如果没有可靠相近物料，直接按类别默认重量给出 `AI估重`。
- 说明中必须披露命中的重量区间、默认重量，以及是否从相近物料修正而来。

### 4.1 电机标准件件价优先规则

以下电机常见标准件或件价型物料，不应因为存在 `AI估重` 就一律打成“估重待复核”状态：

- 轴承
- 内六角 / 外六角螺钉
- 螺母 / 垫圈 / 挡圈 / 卡簧
- 油封 / O 型圈 / 密封垫 / 波形弹簧
- 连接器 / 线束 / 导电环 / 格兰头 / 堵头

处理要求：

- 这类物料优先按“件价逻辑”理解，而不是按重量主导的结构件逻辑理解。
- 若 AI 已返回有效件价，且财务参考价也存在，则可以直接进入“双路线可比”或“价差待复核”判断。
- 只有在 AI 未返回有效价格，或名称/规格明显不足以支持件价估算时，才继续落回 `缺重量 / 待AI报价 / 待补参数`。
- 轴承类尤其要避免被误判成普通机加工件或高精轴类零件。

## 定子与铁芯强规则

### 定子绕组组件规则

当满足以下任一强特征时，必须优先按定子绕组组件理解：

- `物料名` 包含 `定子组件`
- `规格` 包含 `浸漆定子`
- `材质` 包含 `漆包线 / 扁铜线 / 铜线`

此时必须强制映射为：

- 材料：`漆包线材料`
- 工艺：`定子成品工序包`

不允许优先落到通用机械加工、普通注塑件或名称规格兜底路径。

### 定子铁芯规则

当满足以下特征时，必须强制按 `硅钢材料 + 冲压叠压工艺` 处理：

- 物料名包含 `定子 / 铁芯 / 转子铁芯`
- 且材质或规格命中 `35W / 硅钢 / 铁芯 / B30AHV`

此时必须：

- 材料走 `silicon_steel_proxy`
- 工艺走 `冲压叠压`
- 不允许退回通用机械加工或名称规格推断报价

## 材料映射核心规则

优先按公开市场价或合理代理价映射材料：

- 紫铜 / T2 -> 1#铜
- 6063-T5 / 6063-T6 / 6061 -> 铝材代理价
- ZL101 / ZL101+T6 / ZL101+FKM / A356-T6 -> 铸造铝合金代理价
- 35W / 硅钢 / 铁芯 / B30AHV -> 无取向硅钢代理价
- 20CrMnTi / 40Cr / 45钢 / Q235B / SPHC / SPAH440 -> 钢材代理价
- PA66 / PPS / GF 类 -> 树脂材料代理价
- N38UH / 40UH / 48UH / *UH / 钕铁硼 -> 对应磁钢毛坯或成品价，不允许用镨钕金属 1:1 乘成品重量替代

## 工艺估算核心规则

- 按重量计价工艺：`单价(元/kg) × 重量 × 数量`
- 按件计价工艺：`单价(元/件) × 数量`

当前核心工艺口径：

- 冲压叠压：`6.50 元/kg`
- 烧结磁材：`20.40 元/kg`
- 机加工：按复杂度 `20 / 30 / 120 元/kg`
- 注塑：`8.00 元/件`
- 冷镦：`0.80 元/件`
- 压铸：`8.00 元/kg`
- 板金：`6.50 元/kg`
- 焊接：`12.00 元/kg`
- 表面处理 / 装配 / 绝缘 / 浸漆：`10.00 元/kg`
- 定子成品工序包：按定子成品工序打包估算

## 量产三档规则

量产模式必须按年产量分档：

- `<= 5000`
- `5001 - 20000`
- `20001 - 50000`
- `> 50000`

输出必须包含：

- 基准
- 保守
- 激进
- 当前命中的年产量档位

## 输出要求

每次完整报价至少输出：

- 材料映射说明
- 逐行材料费
- 逐行工艺费
- 行总价
- 差异分析
- 假设与推断摘要
- AI推断参考（若触发）

如果系统启用了 AI 税价展示，则仅对 `AI报价` 和 `最终选用价` 做税价拆分展示：

- `AI报价(含税)`
- `AI报价(未税)`
- `AI税额`
- `最终选用价(含税)`
- `最终选用价(未税)`
- `最终选用税额`

约束：

- 财务参考价 / 金蝶采购参考价 保持原始参考口径，不做税价拆分。
- AI 报价总价保持不变，只在内部拆分为未税价与税额。

## 脚本调用推荐流程

默认白名单脚本：

1. `price_bom_xlsx.py`：必跑，负责逐行材料和工艺估算
2. `analyze_pricing_gaps.py`：需要价差复核时跑
3. `format_estimate_workbook.py`：需要导出排版版 Excel 时跑
4. `model_volume_pricing.py`：只有量产模式且给出年产量时跑

执行原则：

- 能由 scripts 算出的价格，优先采用 scripts 结果。
- AI 只负责：
  - 预推断缺失字段
  - 第二轮说明补充
  - scripts 无法形成有效价格时做兜底
- 如果 scripts 已有有效规则价，不要让 AI 直接覆盖 scripts 的主价，只允许补说明或在明确规则下做最终覆盖说明。
