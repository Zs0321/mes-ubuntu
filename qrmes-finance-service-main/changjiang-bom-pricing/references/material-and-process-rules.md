# Material And Process Rules

## 预期列名

优先识别以下字段：

- `产品`
- `产品编码`
- `物料`
- `物料编码`
- `规格`
- `材质`
- `重量(kg)`
- `工艺`
- `含税单价`
- `数量`
- `备注`

允许轻微差异，例如空格、换行或单位写法不同。

## 电机/电控领域默认方向

所有报价默认按电机或电控相关配件理解。

- 当名称或规格存在歧义时，优先识别为电机/电控供应链中的常见配件。
- 名称型物料优先考虑：导电环、法兰、堵头、线束、过滤器、弹簧、密封件、紧固件、绝缘件、接线组件、定转子附件等。

## 复合组件规则

当 `物料编码` 相同且 `物料名称` 相同，但出现多条不同材质行时，按复合组件处理：

- 材料费按各子项分别计价后汇总
- 材质去重拼接显示
- 重量求和
- 规格去重拼接
- 数量保持组件数量
- 工艺按组件共享工艺处理

## 名称规格在线参考规则

当 scripts 未形成有效规则价，但名称/规格足以推断类别时：

1. 先查询公开市场价、公开供应商价格、行业区间价或标准件区间价
2. 记录来源链接、日期、规格范围、是否为同类近似
3. 将在线参考与数量、估重、scripts 上下文一起提供给 AI
4. 最终形成 `名称规格推断报价`

禁止使用 `Excel采购参考 / 金蝶采购参考 / 目标价参考` 作为 AI 报价保底锚点。

## 定子与铁芯规则

### 定子绕组组件

满足以下强特征之一时，必须优先按定子绕组组件处理：

- 物料名含 `定子组件`
- 规格含 `浸漆定子`
- 材质含 `漆包线 / 扁铜线 / 铜线`

映射结果：

- 材料：漆包线材料
- 工艺：定子成品工序包

### 定子铁芯

满足以下特征时，必须按 `硅钢材料 + 冲压叠压工艺` 处理：

- 物料名含 `定子 / 铁芯 / 转子铁芯`
- 材质或规格命中 `35W / 硅钢 / 铁芯 / B30AHV`

映射结果：

- 材料：`silicon_steel_proxy`
- 工艺：`冲压叠压`

## 两阶段 AI 规则

当原始数据缺少 `工艺`、`材质`，或属于重量强相关器件但缺少原始重量时，AI 报价必须拆成两阶段：

1. 第一阶段只做推断：
   - `process_guess`
   - `material_guess`
   - `estimated_weight_kg`
   - `confidence`
2. 第二阶段才做正式报价：
   - 高置信度推断结果可回填给 skills 和 AI 正式报价
   - 若第二阶段返回的工艺/材质与第一阶段不一致，以第二阶段为准

## 材料映射示例

- 紫铜 / T2 -> 1#铜
- 6063-T5 / 6063-T6 / 6061 -> 铝材代理价
- ZL101 / ZL101+T6 / A356 -> 铸造铝合金代理价
- 35W / 硅钢 / 铁芯 / B30AHV -> 无取向硅钢代理价
- 20CrMnTi / 40Cr / 45钢 / Q235B / SPHC / SPAH440 -> 钢材代理价
- PA66 / PPS / GF 类 -> 树脂代理价
- N38UH / 40UH / 48UH / *UH / 钕铁硼 -> 对应磁钢毛坯或成品价

- 冲压叠压：`6.50 元/kg`
- 烧结磁材：`20.40 元/kg`
- 机加工：`20 / 30 / 120 元/kg`
- 注塑：`8.00 元/件`
- 冷镦：`0.80 元/件`
- 压铸：`8.00 元/kg`
- 板金：`6.50 元/kg`
- 焊接：`12.00 元/kg`
- 表面处理 / 装配 / 绝缘 / 浸漆：`10.00 元/kg`
- 定子成品工序包：按定子成品工序打包估算
