📋 使用说明

1️⃣ 用户创建

用户使用群晖账户首次登录时，系统会自动创建本地用户记录，默认为普通用户权限。

2️⃣ 权限管理

管理员可以将普通用户提升为管理员，或将管理员降级为普通用户。

3️⃣ 实时同步

每次登录都会验证群晖账户有效性，确保用户状态实时同步。

4️⃣ 审计日志

所有用户操作和权限变更都会记录详细的审计日志。
